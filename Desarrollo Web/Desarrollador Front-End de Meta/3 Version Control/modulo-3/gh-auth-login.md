# GitHub CLI: Autenticarse eln GitHub desde el terminal

Creado: 28 de septiembre de 2024 8:56

---

El comando `gh auth login` se utiliza para iniciar sesión en GitHub desde la terminal usando la CLI (Interfaz de Línea de Comandos) de GitHub. Permite autenticarte con tu cuenta de GitHub, lo que te habilita a ejecutar comandos de GitHub directamente desde la terminal, como clonar repositorios, crear *pull requests*, revisar issues, entre otros, sin necesidad de ingresar tus credenciales cada vez.

### Pasos generales de uso:

1. Ejecutas el comando: `gh auth login`.
2. Te pedirá que elijas un método de autenticación (navegador web, token personal, etc.).
3. Sigues las instrucciones según el método que elijas.
