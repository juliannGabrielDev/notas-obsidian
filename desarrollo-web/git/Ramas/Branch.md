## Renombrar

```shell
git branch -m main
```

## Crear

```shell
git checkout -b nombre-nueva-rama

git checkout -b nombre-nueva-rama nombre-rama-base
```

## Eliminar

```shell
git checkout otra-rama

git branch -d rama-a-eliminar

git push origin --delete mi-rama-a-eliminar
```

## Listar

```shell
git branch
```
